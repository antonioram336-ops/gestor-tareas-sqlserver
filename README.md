# Gestor de Tareas con SQL Server

Aplicación web CRUD desarrollada con Python, Flask, HTML, CSS y SQL Server.

## Funcionalidades
- Agregar tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas

## Tecnologías utilizadas
- Python
- Flask
- HTML
- CSS
- SQL Server
- pyodbc

## Cómo ejecutar
1. Crear la base de datos en SQL Server.
2. Instalar dependencias:
   pip install -r requirements.txt
3. Ejecutar:
   python app.py
